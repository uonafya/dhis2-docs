16.1. Data Visualizer overview

With the Data Visualizer app, you can create dynamic data analysis and visualizations through charts and data tables. You can freely select content, for example indicators, data elements, periods and organisation units, for an analysis. The app works well over poor Internet connections and generates charts in the web browser. Little data is transferred over the Internet.
