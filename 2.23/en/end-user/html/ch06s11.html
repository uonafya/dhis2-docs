6.11. Embed charts in any web page

Certain analysis-related resources in DHIS 2, like pivot tables, charts and maps, can be embedded in any web page by using a plug-in. You will find more information about the plug-ins in the Web API chapter in the DHIS2 Developer Manual.

To generate a HTML fragment that you can use to display the chart in an external web page:

  1. Click Share > Embed in web page.

    The Embed in web page window opens.

  2. Click Select to highlight the HTML fragment.
