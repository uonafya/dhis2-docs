7.9. Manage map legend sets

You can create your own legend sets to ensure meaningful maps.

Note

Continuous legends are supposed to end and start on the same value, for example 0-50 and 50-80. The GIS app automatically take care of this. Do not try to do this yourself by setting legends to for example 0-50 and 51-80. This will cause a usually unwanted gap in your legend set.

7.9.1. Create a new legend set

  1. Click Legends.

    The Legend sets dialog box opens.

  2. Click Add new.

  3. In the Legend set name box, type the name you want to give the legend set.

  4. Create the legends you want to have in your legend set:

    1. In the Legend name field, type the legend name.

    2. Select Start/end value.

    3. Select Symbolizer.

    4. Click Add legend.

      The legend is added to Current legends.

    5. Repeat until you have created all the legends you need.

  5. Click Create.

    Note

    If your legend set has overlapping legends (for example 0-50 and 40-80) you will not be allowed to proceed. If your legend set has a gap between the legends (for example 0-50 and 60-80) you will get a warning, but are allowed to proceed.

7.9.2. Change a legend set

  1. Click Legends.

    The Legend sets dialog box opens.

  2. Click the edit icon next to the legend set you want to modify.

  3. In the Current legends list, click the edit icon next to the legend you want to modify.

    The Edit legend dialog box opens.

  4. Modify the parameters and click Update.

  5. In the Legend sets dialog box, click Update.

7.9.3. Delete a legend set

  1. Click Legends.

    The Legend sets dialog box opens.

  2. Click the delete icon next to the legend set you want to delete.

  3. Click OK to confirm that you want to delete the legend set.

7.9.4. Assign a legend set to indicator or data element

You can assign a legend set to an indicator or a data element in the Indicator/Data element module. This legend set will then be automatically selected when such an indicator/data element is selected in the GIS.
