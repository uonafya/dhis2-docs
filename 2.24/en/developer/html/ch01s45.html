1.45. SMSCommand

SMSCommands are being used to collect data through SMS. These commands belong to specific Parser Type. Each parser has different functionality.

1.45.1. API End Points

The list of commands can be retrieved using GET.

GET /api/24/sms/commands

One particular command can also be retreived using GET. Where parameter type can be one of the below mentioned parsers

GET /api/24/sms/commands/{commandName}?type=EVENT_REGISTRATION_PARSER

  • KEY_VALUE_PARSER

  • J2ME_PARSER

  • ALERT_PARSER

  • UNREGISTERED_PARSER

  • TRACKED_ENTITY_REGISTRATION_PARSER

  • PROGRAM_STAGE_DATAENTRY_PARSER

  • EVENT_REGISTRATION_PARSER
