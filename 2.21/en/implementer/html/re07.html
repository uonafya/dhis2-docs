Name

dhis2-logtail — Shows the bottom log file in real time. Type Ctrl-C to exit.

Synopsis

/usr/bin/dhis2-logtail {instance name}

Description

Use this tool to show the log of dhis2 instance in real time.

Examples

dhis2-logtail myInstance

See also

dhis2-logview (1).
